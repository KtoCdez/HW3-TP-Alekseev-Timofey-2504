## Инструкция по открытию отчета
1) Для управления проектом используется 'run.sh', поэтому перед проверкой необходимо убедиться, что скрипт имеет права для исполнения: 'chmod +x run.sh'
2) './run.sh build_generator' - сборка образа для генератора
3) './run.sh build_reporter' - сборка образа для аналитики
4) './run.sh run_generator' — запуск генератора
5) './run.sh run_reporter' - запуск аналитики
6) './run.sh report_server' - запуск контейнера для веб-сервера
7) После этого, если вы используете github codespaces, то перейдите в нижней части интерфейса (где терминал) во вкладку порты
8) Найдите в этой вкладке только что развернувшийся сервер на порту 8010
9) Перейдите по сыллке этого сервера в браузер
10) Все, вы в интерфейсе отчета
